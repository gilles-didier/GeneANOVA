GeneANOVA - the LGI Package for Gene Expression Analysis

This software is devoted to expression data analysis and implements the following features:
  • Analyzis of variance (ANOVA)
  • Visualization of data
  • Classification
It has been developped at the laboratory Génome et informatique CNRS UPRESA 8087.  You can address remarks and comments to didier@genopole.cnrs.fr
